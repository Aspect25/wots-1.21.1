# WOTS

A Minecraft 1.21.1 Fabric mod that adds new PLUSHIES FROM MURDER DRONES AND TADC I AM THINKING OF ADDING A LOTTTT

## Dependencies

This mod requires the following to run:

- [Fabric Loader](https://fabricmc.net/use/installer/) 
- [Fabric API](https://modrinth.com/mod/fabric-api)
- Minecraft 1.21.1
- [Geckolib](https://modrinth.com/mod/geckolib/versions)
